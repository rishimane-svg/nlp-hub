# nlp-hub
### An Educational Website for Learning Natural Language Processing

NLP Hub is a multi-page educational website designed to introduce, explain, and explore concepts related to **Natural Language Processing (NLP)**.  
The website is built using **HTML, CSS, and Bootstrap** and is hosted using **GitHub Pages**.

---

##  Project Overview

The aim of this project is to:
- Build a structured, responsive website with multiple linked pages
- Host the website on a public domain
- Analyze user traffic using Google Analytics

This project is developed as part of the **Web Application (WA) Laboratory â€“ Semester VI**.

---

##  Website Structure

The website consists of the following pages:

- **Home (`index.html`)**  
  Introduction to NLP Hub and overview of NLP concepts

- **Learn NLP (`about.html`)**  
  Detailed explanation of NLP, its pipeline, tasks, and modern techniques

- **Datasets (`data.html`)**  
  Information and previews of popular NLP datasets such as IMDB, SQuAD, SST-2, etc.

- **Resources (`resources.html`)**  
  Curated list of NLP courses, tools, libraries, research papers, and communities

- **Contact (`contact.html`)**  
  Contact form and FAQ section for user interaction
